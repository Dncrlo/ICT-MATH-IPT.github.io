Linear Function

A linear function is defined by f(x) = mx + b , where m is the slope and b is the y-intercept, m and b are not both equal to zero. The degree of the function is one and its graph is a line. A linear function is defined by f(x) = mx + b,where m and b are real numbers. Its graph is a line with slope (m) and y-intercept (b).

If m is not equal to 0, then the degree of the function is 1. - Diagonal line
If m = 0 and b is not equal to 0, then the degree of the function is - Horizontal line; function is 0.
If m = 0 and b = 0, then the degree of the function is - no line function is not defined.
